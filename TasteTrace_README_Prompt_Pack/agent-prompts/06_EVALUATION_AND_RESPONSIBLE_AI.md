# Phase 6: Evaluation and Responsible AI

## Objective

Produce reproducible evidence for relevance, diversity, coverage, safety, and product usefulness.

## Offline evaluation

- precision@k and recall@k;
- NDCG or ranking quality;
- catalogue coverage;
- novelty;
- intra-list diversity;
- artist and album repetition;
- popularity exposure distribution;
- baseline comparison;
- sensitivity to model weights.

## Product evaluation

- task completion;
- time to a useful discovery;
- explanation usefulness;
- perceived novelty;
- trust and correction behaviour;
- save and rejection rate.

## Responsible AI

- catalogue representation;
- artist exposure;
- preference privacy;
- sensitive inference risk;
- review-source bias;
- limitations and failure recovery.

## Verification

- no user-study result is claimed without a study;
- all metrics are reproducible;
- model card and privacy documentation match the implementation.
